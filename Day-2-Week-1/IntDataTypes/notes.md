# ***Integer Data Types***
Lesson on how to utilize commonly used integer (whole numbers) data types in C#

<details>
<summary> What are the Four Common Data Types for Integers? </summary>

* **Byte** → 8 bits
* **Short** → 16 bits
* **Int** → 32 bits Most game development projects uses Int
* **Long** → 64 bits

</details>
